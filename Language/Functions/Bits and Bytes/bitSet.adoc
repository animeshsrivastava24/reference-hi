[float]
=== विवरण
सांख्यिक चर का एक सेट (1 को लिखता है)।
[%hardbreaks]


[float]
=== वाक्य - विन्यास (Syntax)
`bitSet(x, n)`


[float]
=== पैरामीटर
`x`: संख्यात्मक चर जिसका बिट सेट करना है। +
`n`: कम से कम महत्वपूर्ण (सबसे दाहिने) के लिए 0 से शुरू होने वाला कौन सा बिट सेट करना है।
